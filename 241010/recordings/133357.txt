[1mDESCRIPTION[0m[m
       The [1mpasswd [22mcommand changes passwords for user accounts. A[m
       normal user may only change the password for their own[m
       account, while the superuser may change the password for[m

[7m Manual page passwd(1) line 1 (press h for help or q to quit)[27m[K[K       associated password validity period.[m
[m
   [1mPassword Changes[0m[m
       The user is first prompted for their old password, if one[m
       is present. This password is then encrypted and compared[m
       against the stored password. The user has only one chance[m
       to enter the correct password. The superuser is permitted[m
       to bypass this step so that forgotten passwords may be[m
       changed.[m
[m
       After the password has been entered, password aging[m
       information is checked to see if the user is permitted to[m
       change the password at this time. If not, [1mpasswd [22mrefuses[m
       to change the password and exits.[m
[7m Manual page passwd(1) line 15 (press h for help or q to quit)[27m[K[K[m
       The user is then prompted twice for a replacement[m
       password. The second entry is compared against the first[m
       and both are required to match in order for the password[m
       to be changed.[m
[m
       Then, the password is tested for complexity. As a general[m
       guideline, passwords should consist of 6 to 8 characters[m
       including one or more characters from each of the[m
       following sets:[m
[m
       ‚Ä¢   lower case alphabetics[m
[m
       ‚Ä¢   digits 0 thru 9[m
[7m Manual page passwd(1) line 29 (press h for help or q to quit)[27m[K[K[m
       ‚Ä¢   punctuation marks[m
[m
       Care must be taken not to include the system default[m
       erase or kill characters.  [1mpasswd [22mwill reject any[m
       password which is not suitably complex.[m

   [1mHints for user passwords[0m[m
       The security of a password depends upon the strength of[m
       the encryption algorithm and the size of the key space.[m
       The legacy [4mUNIX[24m System encryption method is based on the[m
       NBS DES algorithm. More recent methods are now[m
       recommended (see [1mENCRYPT_METHOD[22m). The size of the key[m
       space depends upon the randomness of the password which[m
[7m Manual page passwd(1) line 43 (press h for help or q to quit)[27m[K[K       is selected.[m
[m
       Compromises in password security normally result from[m
       careless password selection or handling. For this reason,[m
       you should not select a password which appears in a[m
       dictionary or which must be written down. The password[m
       should also not be a proper name, your license number,[m
